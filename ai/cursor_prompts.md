# Example Cursor Prompts (Starter Pack)

## Ingest Repo

> “Please ingest the contents of this GitHub repo (URL) and explain your plan for helping me set up my environment step by step for a vibe coding hackathon. I have no coding experience.”

---

## Terminal Basics

> “Guide me to open a terminal and run simple commands like ‘pwd’ and ‘ls’. Explain what each command does in plain language.”

> “Teach me how to switch directories and list files. Show example prompts I can copy/paste.”

---

## Git & GitHub

> “Walk me through installing Git if needed, configuring my name/email, creating a repo, staging a file, committing, and pushing to GitHub.”

> “If I get an authentication issue, explain how to solve it for both macOS and Windows.”

---

## Python / venv

> “Help me install Python, create a virtual environment, and install the Flask package. Provide the exact commands and explanations.”

> “Explain what a virtual environment is and why we need it.”

---

## Node/nvm Setup

> “Help me install nvm, then Node.js version 18, and verify Node/npm are installed.”

> “Initialize a new npm project and install ESLint with recommended defaults.”

---

## Best Practices

> “List simple best practices for naming, typing, and committing that a beginner should follow.”

> “Explain what linting and typing mean for JavaScript/TypeScript and Python.”

---

## Build Prototype

> “Help me scaffold a simple web app that shows a text input and a button. Explain each file that gets created.”

> “Break down the work into subtasks: UI, state management, and backend endpoint.”

---

## Deployment

> “Guide me to deploy the app to Vercel. Explain how to create an account, connect to GitHub, and trigger a first deployment.”

> “If deployment fails, provide debugging steps with plain language explanations.”
